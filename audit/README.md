# ManInTheMiddle — Audit README

Ce dossier `/audit` contient un audit complet du dépôt et une documentation restructurée.

## Contenu
- `repo_audit.md` : inventaire complet + analyse du dépôt.
- `correction_audit.md` : corrections recommandées.
- `todo_audit.md` : plan d’actions.
- `repo_audit.pdf`, `correction_audit.pdf`, `todo_audit.pdf`, `test_audit.pdf` : exports PDF des audits.
- `ARCHITECTURE.md` : architecture technique et flux.
- `INSTALL.md` : installation et prérequis.
- `USAGE.md` : scénarios d’exécution des scripts.
- `CHANGELOG.md` : changelog (synthétique).
- `TESTING.md` : stratégie et commandes de tests.
- `DEBUG.md` : diagnostic et dépannage.
- `test_audit.md` : résultats des tests effectués.

## Remarque
Les scripts du dépôt manipulent le réseau (iptables, hostapd, dnsmasq). Testez dans un environnement contrôlé et avec autorisation explicite.
